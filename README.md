# SimpleUI
UI锦集，持续更新中
打算自己维护一个日常开发会需要用到的UI锦集，方便快(tou)速(lan)完成产品的需求，
每一个UI网上很可能都有轮子，这里只是记录笔者自己的实现思路，细节暂时不予深度优化(^__^)，
例如自定义控件手机旋转的onSaveInstanceState，内存的回收等等，争取每个UI都写一篇博客作为记录
